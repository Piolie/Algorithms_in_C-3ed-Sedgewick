/* ---------------------------------------------------------------------
                             EXERCISE 2.25
------------------------------------------------------------------------
Prove that N/(N + O(1)) = 1 + O(1/N).
--------------------------------------------------------------------- */

             N
  f(N) = ----------
          N + O(1)

          N + O(1)       O(1)
       = ---------- - ----------
          N + O(1)     N + O(1)

                O(1)
       = 1 - ----------
              N + O(1)

                O(1)
       = 1 - ----------
              N + O(1)

       = 1 - O(1/N)

The last step uses N + O(1) ~ N and O(1)/N = O(1/N).
